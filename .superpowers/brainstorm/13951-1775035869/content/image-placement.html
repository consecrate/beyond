Where should images live in PlayDeck slides?

Images can serve different purposes depending on the slide type. Which placement(s) make sense for your app?

A

Show slides — full background or inline image

Static content slides (like lecture slides) can have a hero image or an image embedded in the markdown body — used to explain concepts visually.

B

Question / Poll slides — image as part of the question prompt

"Look at this image — which answer is correct?" The image appears above the question text, giving students visual context to answer from.

C

Answer options — images as choices

Each answer option is an image instead of (or alongside) text. Students pick the right picture. Great for visual identification questions.

D

All of the above — images anywhere

Maximum flexibility: authors can put images in show slides, question prompts, and answer options. Handle it uniformly as "markdown supports images".
